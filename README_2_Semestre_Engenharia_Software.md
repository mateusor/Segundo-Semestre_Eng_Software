# 🎓 2º Semestre --- Engenharia de Software

Este repositório reúne os conteúdos, projetos, exercícios e materiais
desenvolvidos durante o **2º semestre do curso de Engenharia de
Software**.

O objetivo é organizar os estudos de forma clara e criar um **portfólio
acadêmico no GitHub**, registrando a evolução nas disciplinas do curso.

------------------------------------------------------------------------

# 📚 Disciplinas do Semestre

Durante este semestre foram estudadas as seguintes disciplinas:

-   📊 **Estudos de Banco de Dados I**
-   💻 **Algoritmos e Linguagens de Programação**
-   🧩 **Projeto Integrador II**
-   🌐 **Introdução às Tecnologias para Web**
-   📋 **Engenharia de Elicitação de Requisitos**
-   ⚙️ **Fundamentos de Engenharia de Sistemas**
-   🛠 **Métodos de Engenharia de Software**
-   🧠 **Estrutura e Recuperação de Dados I**
