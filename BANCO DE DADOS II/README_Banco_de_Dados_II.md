# 📚 Banco de Dados II

Repositório dedicado aos conteúdos, exercícios e projetos desenvolvidos
na disciplina **Banco de Dados II** durante o curso de **Engenharia de
Software**.

## 🎯 Objetivo

Este repositório foi criado para organizar os materiais e projetos
relacionados à disciplina, incluindo:

-   Modelagem de dados
-   Banco de dados relacionais
-   SQL
-   Normalização
-   Projetos práticos de banco de dados

## 📂 Estrutura do Repositório

Banco-de-Dados-II/ │ ├── Projetos/ │ └── Concessionaria-BD/ │ ├──

## 🚗 Projeto Final

O projeto final da disciplina consiste na criação e modelagem de um
banco de dados para uma **Concessionária de Veículos**.

O projeto inclui:

-   Modelo Conceitual
-   Modelo Lógico
-   Script SQL
-   Estrutura de tabelas e relacionamentos

Acesse o projeto em:
